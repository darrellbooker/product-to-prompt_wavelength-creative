# 🚀 Prompt to Product — Wavelength Creative Operations Dashboard

Built live during the **Prompt to Product Workshop Series** by [Darrell Booker](https://linkedin.com/in/darrellbooker), this is a custom business operations dashboard created entirely through conversational AI prompts using GitHub Spark. No traditional coding required.

## What Is This?

This is a fully functional operations system for a creative agency, built in 45 minutes using nothing but natural language prompts. It includes four integrated components:

| Component | What It Does | What It Replaces |
|-----------|-------------|-----------------|
| **Social Media Scheduler** | Plan, draft, and track posts across platforms with filtering and multi-platform support | Later, Hootsuite, Buffer |
| **Email Campaign Manager** | Compose campaigns with a rich text editor, manage templates, track performance | Mailchimp, ConvertKit, HubSpot |
| **Staff Culture Builder** | Employee profile cards, 1-on-1 tracking, development goals, and recognition | 15Five, Lattice, Google Forms |
| **Campaign ROI Dashboard** | Track spend, revenue, and auto-calculated ROI/ROAS with CSV export | Google Sheets + manual reporting |

## 🛠 About the Build

This project demonstrates **"vibe coding"** — building software through iterative natural language conversation with AI. Every component was created by describing what was needed in plain English, reviewing the output, and refining through follow-up prompts. The prompts used are included in the workshop deck and in the `sparks/` directory.

GitHub Spark leverages frontier AI models (including GPT and Claude) under the hood. It's not its own model — it's a platform that packages the best AI available into a tool designed specifically for building apps.
